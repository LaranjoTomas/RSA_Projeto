# Dashboard

## Como Executar

1. **Iniciar o Backend**:
   - Na pasta `dashboard` executa:
     ```bash
     python3 server.py
     ```

2. **Iniciar o Frontend**:
   - Na pasta `dashboard/app` executa:
     ```bash
     npm run start
     ```
